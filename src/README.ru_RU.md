## Использование

### Использование калькулятора в консоли

1. Запуск

    ```shell
    php artisan calculator:sum [<operand1> [<operand2>]]
    ```

    Параметры:
    
    * operand1 - Первый параметр
    * operand2 - Второй параметр

    Ошибки:

    |Значение|En|Ru|
